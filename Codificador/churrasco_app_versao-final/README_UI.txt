Churrasco App - Swing Frontend Scaffold
=======================================

O pacote de UI em Java Swing foi adicionado em:
  - src/ui/swing/MainSwingLauncher.java
  - src/ui/swing/frames/*.java
  - src/ui/swing/panels/*.java

Como rodar no IntelliJ:
  1. Importe o projeto como 'Empty Project' ou 'Java project'.
  2. Certifique-se de que o diretório 'src' está marcado como Sources Root.
  3. Compile o projeto.
  4. Rode a classe 'ui.swing.MainSwingLauncher' (Run -> Edit Configurations -> Main class).

Observações:
  - Este é um scaffold inicial com telas e navegação. Integrações com o back-end
    (controllers, DAOs e persistência) ainda precisam ser conectadas aos pontos
    de chamada no LoginFrame, MainFrame and panels.
  - Eu deixei mensagens informando onde implementar a lógica (comentários e diálogos).
